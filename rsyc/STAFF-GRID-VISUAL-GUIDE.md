## Responsive Breakpoints

### Mobile (< 768px)
- Single column layout
- Full-width filter selects stacked vertically
- Touch-friendly button sizes
- Simplified spacing

Example: `grid-template-columns: 1fr`

### Tablet (768px - 1024px)
- 2 columns for staff cards
- 2x2 filter grid layout
- Slightly larger cards
- Optimized padding

Example: `grid-template-columns: repeat(2, 1fr)`

### Desktop (> 1024px)
- 3-4 columns for staff cards
- 3-column filter layout (state, city, type)
- Larger images
- Spacious layout

Example: `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`

## Color Scheme

### Staff Card Colors
- **Card Background**: White (#ffffff)
- **Text (Name)**: Dark gray (#333333)
- **Title Text**: Teal (#20B3A8) - matches RSYC brand
- **Location Text**: Medium gray (#666666)
- **Bio Text**: Darker gray (#555555)
- **Icon Color**: Light gray (#999999)
- **Background Section**: Light gray (#f8f9fa)

### Filter/Button Colors
- **Filter Background**: Light gray (#f8f9fa)
- **Clear Button**: Gray (#6c757d)
- **Clear Button Hover**: Darker gray (#5a6268)
- **Select Border**: Light border (#dee2e6)
- **Option Text**: Dark (#212529)

## Typography

### Font Stack
Primary: sans-serif (inherited from Bootstrap)

### Font Sizes
- **Card Title (Name)**: 1.1rem / 18px, Bold
- **Card Subtitle (Title)**: 0.9rem / 14px, Medium Weight, Teal
- **Card Text (Center)**: 0.85rem / 14px, Regular
- **Card Text (Location)**: 0.85rem / 13px, Regular
- **Card Bio**: 0.9rem / 14px, Regular, Line-height 1.5
- **Filter Labels**: 0.9rem / 14rem, 500 weight
- **Section Heading**: 2rem / 32px, Bold

## Spacing

### Card Spacing
- Grid gap: 24px (all directions)
- Card padding: 1.25rem (20px) all sides
- Image height: 250px
- Minimum card width: 280px

### Section Spacing
- Section padding: 60px vertical, 20px horizontal
- Container max-width: 1400px
- Center heading margin-bottom: 3rem
- Subheading margin-bottom: 0.75rem
