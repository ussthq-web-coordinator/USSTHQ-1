Comparison Rules Applied
Data Normalization Rules:

  • URL normalization: All http:// URLs are converted to https://

Data Matching/Keying:
  • GDOS ID Matching: GDOS records are matched to Zesty records using the GDOS ID field
  • GDOS Source: gdos.id (string identifier)
  • Zesty Source: Column1.content.gdos_id (flattened JSON key)
  • Matching Process: Only published GDOS records with matching Zesty gdos_id are compared
  • Unmatched Records: GDOS records without Zesty matches are excluded from comparison
